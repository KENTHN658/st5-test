แบบประเมินอารมณ์ของคุณวันนี้!!!

ให้คุณลองประเมินตนเอง โดยให้คะแนน 0-3 ที่ตรงกับความรู้สึกของคุณ

คะแนน 0 : แทบไม่มี

คะแนน 1 : เป็นบางครั้ง

คะแนน 2 : บ่อยครั้ง

คะแนน 3 : เป็นประจำ

โดยเราจะมีอาการที่เกิดขึ้นให้คุณได้ทำแบบทดสอบ เพื่อทดสอบอารมณ์ของตนเอง ว่าวันนี้คุณมีอารมณ์อย่างไร โดยผลการประเมินเราจะสรุปจากคะแนน

คะแนน 0-4 : เครียดน้อย

คะแนน 5-7 : เครียดปานกลาง

คะแนน 8-9 : เครียดมาก

คะแนน 10-15 : เครียดมากที่สุด

โดยคะแนนจะออกมาในรูปของ Emoticon บอกอารมณ์ของเรา และมีการแสดงเป็นกราฟเพื่อบ่งบอกถึงการเปลี่ยนแปลงทางอารมณ์ของคุณ รวมถึงเรายังมีให้คุณสามารถบันทึกไดอารี่เกี่ยวกับคุณในวันนั้น แล้วให้คุณเลือกว่าคุณอยากแชร์เรื่องราวของตัวเองให้ผู้อื่นรับรู้(Public) หรือ เก็บเป็นความลับสำหรับคุณเอง(Private)
